Nerd List: Portland Nerd Events

A high-performance, retro-inspired static directory for nerd events in Portland. Built with Eleventy and a custom TypeScript data pipeline.

ğŸ“ Project Structure

  • docs/rk-events.md: Source of Truth. Edit this file to add/update events.
  • src/: Core logic and adapters.
    • adapters/MarkdownAdapter.ts: The parser logic for the event markdown.
    • utils/normalize.ts: Data cleaning and ID generation.
    • css/style.css: The retro design system.
  • _data/: (Ignored) Canonical JSON data generated by the scraper.
  • _includes/: Nunjucks layout templates.
  • .github/workflows/deploy.yml: Automated CI/CD for GitHub Pages.

ğŸ¤– CI/CD

Deployment is automated via GitHub Actions. On every push to main:

  1. It runs the scraper to sync the latest markdown changes.
  2. It builds the static site with the correct GitHub Pages path prefix.
  3. It deploys the resulting /_site directory to the gh-pages branch.
